on finally ricing my setup (sort of)

I never riced my Arch Linux setup, but I always wanted to. Now that AI exists, it was relatively facile to vibe rice my website.

For years, I ran Arch Linux with i3wm. Thanks for setting me up https://fabilab.org/. But I never riced it. Too busy analyzing sequencing data.

Enter my good buddy Claude Code. And suddenly the idea of "vibe ricing" became possible. You know — just describe the rice, Claude figure out the CSS, the layout, the terminal colors.

So instead of ricing my dead Linux setup (I don't use it anymore — I'm on a Mac so I can run Ableton Live), I riced my website instead.

Here it is, an i3wm-inspired portfolio site. Terminal aesthetics. ASCII art. unix p*rn. Status bar replete with fake system stats. So there's no dotfiles, but get in touch if you want the github repo :D
